Novos comandos para a tartaruga!

Vamos tentar criar um mapa de labirinto usando uma tartaruga!

Mas primeiro, você precisa aprender alguns comandos novos!

1. Como fazemos a tartaruga parar de desenhar? Existe um comando para isso:

t.up()

Note que deixamos os colchetes vazios. O comando t.up() diz à tartaruga que não há necessidade de deixar rastros, o que significa que todos os seus movimentos permanecerão despercebidos.

2. Como obtemos o rastro da tartaruga agora? Há um comando para isso:

t.down()

E novamente, os parênteses estão vazios!

3. Para mudar a cor da tartaruga (assim como a cor do seu traço), usaríamos o comando t.color()! Aqui está como ele é usado:

t.color("red")

Temos que garantir que a cor esteja entre aspas!
