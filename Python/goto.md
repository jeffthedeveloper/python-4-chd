Pergunta: Quais comandos você usaria?

Qual é a outra opção?

Para tornar o processo mais prático e rápido, você pode usar:
coordenadas e o comando .goto()

Coordenadas são os pontos onde nossa tartaruga pode se mover. A coordenada X é responsável por mover a tartaruga horizontalmente (esquerda e direita). A coordenada Y é responsável por mover a tartaruga verticalmente (para cima e para baixo).

As coordenadas são úteis porque podemos enviar a tartaruga para qualquer ponto do desenho usando o comando .goto()!

Dentro dos parênteses, especificamos dois números separados por vírgula:
o primeiro é um ponto no eixo X o segundo é um ponto no eixo Y
