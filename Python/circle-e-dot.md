Os novos comandos da tartaruga

Para completar essa tarefa, precisamos aprender sobre outras duas ferramentas: os comandos circle() e dot()!

O comando circle() permite que a tartaruga desenhe um círculo vazio de um tamanho específico. O tamanho é colocado dentro dos parênteses, por exemplo:

t.circle(100) # círculo com um diâmetro de 100 pixels
t.circle(50) # círculo com um diâmetro de 50 pixels

O comando dot() funciona de uma maneira parecida. A diferença é que o círculo é colorido, mas o tamanho também é colocado dentro dos parênteses.

t.dot(50)  # ponto colorido com um diâmetro de 50 pixels
