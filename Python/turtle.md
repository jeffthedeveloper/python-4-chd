

Como criar seu passe usando código?

Lembra que dissemos que um computador é um robô? Nosso robô vem com uma tartaruga artista especial que seguirá nossos comandos em sequência! Precisamos compor instruções para que ela crie um passe para você!

3 passos para criar um passe

Etapa 1. Insira o código
Digite o seguinte código para começar a usar a tartaruga:

import turtle

t = turtle.Turtle()
t.shape("turtle")

Etapa 2. Execute o código
Para executar o código e ver a tartaruga, pressione o botão EXECUTAR

Etapa 3. Defina uma direção
Agora que a tartaruga foi criada, você pode dar uma direção a ela com os seguintes comandos:

t.forward(100) – o movimento para frente da tartaruga (a distância em pixels é mostrada entre parênteses).

Tarefa adicional ⭐

Você terminou antes de todo mundo? Bom trabalho!

Se tiver tempo, tente criar um passe sem linhas extras, você pode fazer isso com os comandos t.up() e t.down()

t.up() é um comando que permitirá que você levante a tartaruga e, até que você a abaixe, todos os comandos subsequentes serão executados sem contorno.
t.down() é um comando que permite abaixar a tartaruga se você a levantou, para que o contorno continue.
