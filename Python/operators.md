Operadores do Python

Para resolver os enigmas do baú, você precisa aprender algo novo - operadores! Aqui estão os sinais para operações comuns:

    Adição: +
    Subtração: -
    Multiplicação: *
    Divisão: /
    Combinação: ()

Esses operadores ajudarão você a fazer operações aritméticas com números e strings, mas nem todos eles podem interagir com strings (uma string é um texto que você coloca entre aspas):

Somente +, * e () - podem ser aplicados a strings

O restante exibirá um erro

print(3 * "Hello")

Este programa irá gerar...

HelloHelloHello

Agora, o programa com este código...

print("Hello" + " " + "world")

Irá imprimir isto

Hello world

Note que um espaço também é um caractere! Portanto, ele também deve ser colocado entre aspas!
