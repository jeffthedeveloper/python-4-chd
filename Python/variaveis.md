

Variáveis



Imagine que você mantém um caderno onde anota os endereços dos seus amigos.

Você dá um nome a cada registro, por exemplo: "Endereço do João", "Endereço da Mary". Quando você precisa encontrar o endereço do João, você apenas procura pela entrada "Endereço do João"!

    O nome da entrada é o nome da variável

    O endereço dentro do registro é o valor da variável

Vamos escrever isso em código, tudo bem?
Para criar uma variável, usamos o operador de atribuição =

# Criar variáveis ​​para endereços de amigos
joão_address = "Rua São Paulo, nº 10"
maria_address = "Rua Pernambuco, nº 5"

# Emitindo os endereços
print("Endereço do João:", joão_address)
print("Endereço da Maria:", maria_address)



Vamos entender como definir nomes corretos para variáveis. A sintaxe (a palavra para as regras de escrita de código) é muito importante!

    As variáveis ​​sempre começam com uma letra minúscula

    É comum usar sublinhados para escrever variáveis ​​multipalavras

    Variáveis ​​não podem começar com números

    Uma última regra muito importante - as variáveis ​​devem ser nomeadas logicamente! Se você tiver informações sobre o nome dentro da sua variável, chame-a de 'nome'
